## ✨ About

QuickNote is a lightweight Windows application that opens instantly with a custom keyboard shortcut, allowing you to capture ideas in seconds.

No distractions.
No unnecessary menus.
Just write and keep working.

---

## 🚀 Features

- ⚡ Instant popup with a hotkey
- 📝 Fast note creation
- 💾 Automatic saving
- 📂 Local note storage
- 🎨 Clean modern interface
- 🪶 Lightweight and responsive

## ⚙️ Installation

```bash
git clone https://github.com/yourusername/QuickNote.git

cd QuickNote

pip install -r requirements.txt

python src/main.py
```
